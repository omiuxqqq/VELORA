# VELORA | Online Store

Готовый каркас интернет-магазина VELORA для Vercel + Next.js.

## Страницы
- `/` главная
- `/catalog` каталог
- `/product/1` карточка товара
- `/cart` корзина
- `/checkout` оформление заказа
- `/login` вход
- `/register` регистрация
- `/profile` личный кабинет
- `/reviews` отзывы
- `/admin` админ-панель
- `/admin/products` товары администратора
- `/admin/orders` заказы администратора

## Supabase
SQL для базы находится в `supabase/schema.sql`.
